FAQ

Common questions about healthcare interoperability.

Which EHR systems can you integrate with?

We have production experience with PointClickCare, Epic, Gehrimed, Netsmart, and various custom EHRs. If an EHR has an API or supports HL7/FHIR, we can integrate with it.

How long does an integration project take?

Typical integration projects take 2-4 months depending on complexity, EHR vendor cooperation, and scope. Simple point-to-point integrations can be faster; enterprise-wide interoperability platforms take longer.

Do you support both HL7 v2 and FHIR?

Yes. Most healthcare organizations have a mix of legacy and modern systems. We implement the right standard for each integration—HL7 v2 for legacy, FHIR for modern APIs, and translation layers when needed.

How do you handle data security?

All integrations are HIPAA-compliant with encryption in transit and at rest, role-based access controls, and comprehensive audit logging. We've been doing this since 2007.

Can you integrate with systems you haven't worked with before?

Absolutely. Our team has deep experience with healthcare data standards—if a system supports standard protocols or has an API, we can integrate with it. We'll assess feasibility during discovery.
