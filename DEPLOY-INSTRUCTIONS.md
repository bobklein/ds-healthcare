## Deployment Steps

### Option A: GitHub Web Upload
1. Go to https://github.com/bobklein/ds-healthcare
2. Click "Add file" → "Upload files"
3. Drag ALL files from this folder
4. Commit message: "Apply design standards + GTM positioning"
5. Click "Commit changes"

### Option B: Git Command Line
```bash
cd ~/ds-healthcare
# Copy all files from this package to repo root
cp /path/to/production-deploy/* .
git add .
git commit -m "Apply design standards + GTM positioning to all pages"
git push origin main
```
