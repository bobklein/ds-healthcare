```html

     alt="Description" 
     class="rounded-xl"
     loading="lazy">
```

### Image with Hover Effect (for cards)
```html
<div class="h-48 overflow-hidden">
    <img src="[URL]" alt="[Description]" 
         class="w-full h-full object-cover object-top group-hover:scale-105 transition duration-300"
         loading="lazy">
</div>
```

### Hero Image with Overlay
```html
<div class="relative">
    <img src="[URL]" alt="[Description]" class="w-full h-auto rounded-2xl">
    <div class="absolute inset-0 bg-gradient-to-t from-dsBlack/50 to-transparent rounded-2xl"></div>
</div>
```
